# Tools and Technologies Used

## Programming Language
- Python 3.8+

## Data Analysis & Machine Learning
- pandas — data loading, cleaning, and manipulation
- numpy — numerical computation
- scikit-learn — Decision Tree, Random Forest, KNN, preprocessing, train/test split, metrics
- xgboost — gradient-boosted classification model
- matplotlib / seaborn — exploratory data visualization

## Model Persistence
- joblib — saving/loading the trained model, scaler, and column schema

## Web Application
- Flask — backend web framework and routing
- HTML5 / CSS3 — front-end structure and styling
- Jinja2 templating — dynamic rendering of prediction results

## Development Environment
- Jupyter Notebook / Anaconda Navigator — data exploration and prototyping
- VS Code / any IDE — application development

## Deployment (Planned)
- IBM Cloud — scalable hosting for the Flask application
- Git & GitHub — version control and source management

## Dataset
- Historical meteorological data: annual rainfall, seasonal (monsoon) rainfall,
  cloud visibility, humidity, temperature, and river discharge readings across
  multiple regions.
